IESCO duplicate bill guide

Use this when you need the real bill copy, due date, or payable amount from the public bill portal.

What you need first

The public duplicate-bill workflow normally starts with a 14-digit reference number or a 10-digit customer ID. If you already have either one, use the public PITC portal instead of relying on an estimate alone.

What the duplicate bill gives you

  • billing month
  • due date
  • meter readings
  • current payable amount
  • line items already printed on the official bill copy

How it differs from the homepage calculator

The homepage tool is for estimating and cross-checking. The duplicate-bill portal is for retrieving the actual bill record. If you already have the real bill, use this site to understand it better. If you do not have the bill at all, go to the duplicate-bill portal first.

Open duplicate bill portal Go to the public PITC page linked by IESCO to view or print the current bill. Need the number first? Open the reference-number guide if the duplicate-bill portal is not usable because the account number is missing.
Duplicate bill access is kept on its own page because users searching for a bill copy are not always trying to estimate charges.
